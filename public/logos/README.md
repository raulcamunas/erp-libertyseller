# Logos

Coloca aquí el logo de Liberty Seller Hub.

## Uso en Next.js

Una vez que coloques tu logo aquí, puedes usarlo en tus componentes así:

```tsx
import Image from 'next/image'

// Ejemplo de uso
<Image 
  src="/logos/tu-logo.png" 
  alt="Liberty Seller Hub" 
  width={200} 
  height={50}
/>
```

## Formatos recomendados

- PNG con fondo transparente (preferido)
- SVG (escalable, mejor calidad)
- JPG/JPEG (si no necesitas transparencia)

## Tamaños sugeridos

- Logo principal: 200x50px o similar
- Logo pequeño (favicon): 32x32px o 64x64px
- Logo grande (hero): 400x100px o similar
